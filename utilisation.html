Brève explication de Yayo

Yayo est une application mobile innovante et efficace spécialisée dans la gestion des points de vente
Yayo vous permet de créer gratuitement votre point de vente et de suivre de près ses activités grâce aux meilleures technologies du moment
Vous pouvez ainsi gérer à partir des vos articles jusqu'à vos dettes en passant par vos ventes et vos depenses, tout ça en tant réel et dans des fractions de sécondes
Yayo vous génére aussi les documents financiers sous format pdf partageable, imprimable et télechargable
Yayo vous offre aussi une ouverture au monde en vous permettant la publication et la discussion de vos produits dans la partie Circulaires et en vous faisant parvenir aux annonces des entreprises et organisations dont les activités peuvent vous être benefiques
Yayo est sécurisé et très facile à utiliser, pas besoin de compétences techniques ni de matériel spécifique, juste votre SMART phone et une petite connexion internet
Vous pouvez aussi faire la gestion de votre propre compte et de votre point de vente si vous en êtes administrateur (en posseder le code admin)
Quant au paiement, Yayo facture 1 $ par point de vente par mois pour son service dans sa version actuelle et la restriction à ceux qui ne payent pas c'est le non accès aux rapports,factures et notes de crédit sous format PDF

De la création de point de vente

Voici la liste complete d'elements que Yayo demande lors de la création d'un point de vente:

  1. Le logo (facultatif) : C'est l'image qui apparait comme logo sur les factures, les notes de credit, la page d'accueil de la partie Caisse et Gestion, les rapports, les fiches de stock, ect. S'il n'est pas renseigné, le logo de Yayo sera affiché partout où il devrait être affiché dans l'application et sur les documents
  2. La catégorie (obligaoire) : C'est le genre du point de vente que vous créez, leur liste est exhaustive pour chaque version, Yayo filtre le depôt d'articles et les annonces d'un point de vente selon sa catégorie
  3. Le nom (obligatoire) : C'est le nom qui apparait en tête de la page d'accueil, sur les factures, sur les notes de credit, sur les raaports, sur les fiches de stock, etc. Le nombre maximal de caractères d'un nom c'est 25
  4. Le code admin (obligatoire) : C'est juste un code de votre choix qui vous permettra dans la suite d'avoir beaucoup d'autorisation par rapport aux autres membres de votre point de vente, Si vous l'oubliz, vous pouvez le récuperer via le mail admin à l'entrée de l'esapce de gestion
  5. Le taux d'échange et les 2 devises (obligaoire) : C'est grâce à ces elements que Yayo vous gérére des rapports bien calculé, la 1iere devise choisie est la devise principale et le 2ieme la devise sécondaire. Les 2 dévises ne seront jamais modifiées
  6. Le phone admin (obligatoire) : Le numéro de téléphone avec le code du pays grâce auquel Yayo contacte votre point en cas de besoin
  7. Le mail admin (obligatoire) : C'est à ce adresse email que Yayo envoit un message en cas de confirmation de paiement, de mauvais paiement, et d'autres nouvelles
  8. La position (obligatoire) : Yayo ne récupere que votre pays pour mieux filtrer les annonces et les circulaires à proposer aux membres de votre point de vente

De tous ces elements, seules les devises du taux d'échange ne changent jamais dans le futur du point de vente
Une fois le point de vente créé, Yayo envoit la clé du point de vente à son créateur dans l'application et sur Email au mail admin et desormais le créateur est consideré garant de tous ceux qui s'inscriront et se connecteront via la clé lui envoyée
La clé doit être gardé jalousement parceque Yayo n'a pas encore integré la récuperation de clé de point de vente sa version actuelle
Yayo considere que le créateur du point de vente a l'âge lui autorisant de posséder un point de vente dans son pays
Yayo crypte le code admin du point de vente avant de le stoquer dans la base de données

De la création de compte utilisateur

Voici la liste complete d'elements que Yayo demande lors de la création d'un compte utilisateur:

  1. L'image de profile (obligatoire) : C'est l'image qui apparait sur les circulaires et notifications de messaages dont vous êtes l'auteur, et aussi sur la tête de la page d'accueil
  2. Le nom (obligatoire) : C'est le nom que apparait comme auteur aux ventes, depenses, dettes, circulaires, messaages, etc dont vous êtes auteur
  3. Le mot de passe (obligatoire) : C'est juste un mot de passe de votre choix qui lui permettra dans la suite de vous connecter dans n'importe quel téléphone contenant l'application Yayo, il ne doit pas être oublié car pour le récuperer il vous sera demandé de donner votre adresse email à l'admin de votre point de vente et ce dernier actualisera votre compte pour que Yayo vous envoie un mot de passe provisoir sur votre adresse email
  4. La clé du point de vente (obligaoire) : C'est la clé du point de vente sur lequel vous souhaitez vous inscrire

Dans Yayo, le nom d'utilisateur est unique par point de vente
Yayo considere que le créateur du compte utilisateur a l'âge lui autorisant de travailler dans un point de vente dans son pays et qu'il a trouvé légalement la clé du point de vente auquel il s'inscrit
Yayo crypte le mot de passe de l'utilisateur avant de le stoquer dans la base de données

De la connexion de l'utilisateur

Voici la liste complete d'elements que Yayo demande lors de la connexion d'un utilisateur:

  1. Le nom (obligatoire) : C'est le nom sous lequel l'utilisateur s'etait inscrit
  2. Le mot de passe (obligatoire) : C'est le mot de passe avec lequel l'utilisateur s'etait inscrit
  3. La clé du point de vente (obligaoire) : C'est la clé du point de vente auquel l'utilisateur veut se connecter

De la gestiion des articles d'un point de vente

Tous les membres d'un point de vente ont le même pouvoir aux articles de ce dernier
chaque membre d'un point de vente, peut y ajouter un ou plusieurs articles, y voir les localisation des articles, y vendre les articles, y ravitailler les articles, y modifier les articles, y dupliquer les articles et y supprimer les articles
Chaque article doit posseder obligatoirement une image, un nom, une quantité, une unité, un prix de vente unitaire, une dévise de vente, un emplacement dans le point de vente et facultativement une date d'expiration
Après création d'un article, on ne peut jamais en changer la dévise et l'unité
L'article est ajouté dans un point de vente soit à partir du depôt d'articles soit en completant le formulaire d'ajout soit en dupliquant un article existant
Les articles ajoutés depuis le depôt d'articles, n'ont pas de date d'expiration et d'emplacement et ont pour quantité 10 unité
Lors de la vente d'un article, Yayo alerte et demande à l'utilisateur sa permission avant d'enregister un vente à perte selon le prix de vente de l'article et la quantité mentionnée par l'utilisateur
Sur la liste affiché, les articles se suivent par ordre de date d'expiration les date d'expiration des articles expirés s'affichant en couleur rouge
Les actions effectuées sur les articles sont repercutées en temps réel à tous les membres d'un même point de vente

De la gestion des ventes d'un point de vente

Les ventes sont créées soit depuis la partie Articles (Vente d'un article) ou soit depuis la Dette (paiement d'une dette)
Dans la partie Ventes, la liste de ventes s'affiche par ordre decroissant du moment de création
Les vente dont l'article a été supprimé, s'affiche avec le logo de Yayo Chaque vente possede obligatoirement l'article vendu, la quantité vendue, la dévise, le montant vendu, la date et l'heure ,l'auteur de la vente et facultativement le nom du client
Yayo genere automatiquement le benefice de la vente le prix de vente unitaire lors de la création d'une vente, c'est le benefice ainsi generé qui est renvoyé lors de la generation des rapport
Chaque membre de l'espace peut voir toutes les vente et les inclure dans la facture mais seul l'auteur d'une vente peut la modifier ou la supprimer
Lors de la modification d'une vente, Yayo alerte et demande à l'utilisateur sa permission avant d'enregister un vente à perte selon le prix de vente de l'article et la quantité mentionnée par l'utilisateur

De la gestion des dépenses d'un point de vente

Chaque dépense possede obligatoirement le motif, la dévise, le montant, la date, l'heure et l'auteur
Toutes les dépenses sont créées explicitement depuis la partie Depense sauf les dépenses de type ravaillement qui sont créer losrqu'un utilisateur ravitaille un article
Dans la partie Depenses, la liste de dépenses s'affiche par ordre decroissant du moment de création
Chaque membre de l'espace peut voir toutes les dépenses mais seul l'auteur d'une dépense peut la modifier ou la supprimer
A la fin de la facture se trouve un messaage de remerciement de Yayo au point de vente

De la gestion des dettes d'un point de vente

Chaque dette possede obligatoirement le nom du client ou contractant , l'article, la date, l'heure, l'auteur, la quantité, le montant
Yayo genere automatiquement le prix de vente unitaire et stock le prix d'achat de l'article lors de la création d'une dette, c'est le prix d'acaht ainsi stocké qui est utilisé pour determiner le benefice lors du paiement d'une dette
Dans la partie Dettes, la liste de dettes s'affiche par ordre decroissant du moment de création
Tous les membres du point de vente peuvent voir les dettes mais seul l'auteur d'une dette peut en enregister le paiement ou la supprimer
Tous les membres du point de vente peuvent génerer a note de credit les clients
A la fin de la note de crédit, se trouve le message de remerciement de Yayo au point de vente

De la Caisse et la Gestion d'un point de vente

Cette partie est reservée aux administrateurs du point de vente donc l'utilisateur ne peut pas y acceder sans connaitre le code admin du point de vente auquel il est connecté
Dans cette partie l'on trouve les elements suivants en rapport directe avec la caisse et le gestion du point de vente

  1. Rapport périodique : Un fichier PDF contentant les entrées (avec benefices) sous forme de ventes, les sorties sous forme de dépense et les dettes dans un intervalle de dates choisies par l'utilisateur, il est aussi demandé à l'utilsiateur la dévise sous laquelle il prefere avoir le rapport périodique et les calculs sont faits selon le taux d'échange du point de vente au moment de demande du rapport périodique
  2. Rapport journalier : Un fichier PDF contentant les entrées (avec benefices) sous forme de ventes, les sorties sous forme de dépense et les dettes d'une date choisie par l'utilisateur, il est aussi demandé à l'utilsateur la dévise sous laquelle il prefere avoir le rapport journalier et les calculs sont faits selon le taux d'échange du point de vente au moment de demande du rapport journalier
  3. Fiche de stock : Un fichier PDF contentant tous les articles du point de vente et leur valeur totale en achat et en vente
  4. Membres du point de vente : Liste de tous ceux qui ont créés leurs comptes utilisateur avec la clé de l'actuel point de vente, l'administrateur peut supprimer tous les membres indésirables
  5. Parametres du point de vente : Liste de tous les parametres de l'actuel point de vente avec la possibilité d'en modifier les modifiables

A la fin du rapport périodique, journalier et de la fiche de stock, se rouve le messaage de remerciement de Yayo au point de vente

De l'abonnement au service de Yayo

Dans sa version actuelle, Yayo facture 1 $ par point de vente par mois
Chaque membre du point de vente peut voir les différents paiements déjà effectués par le point de vente
Tous les membres du point de vente peuvent enregister des paiements et pour chaque paiement validé ou rejeté, Yayo envoit un email de confirmation à l'email admin du point de vente
Yayo met à jour les abonnement des points de vente au debut de chaque jour (8h00 heure de Goma en RDC), cette operation etant semi manuelle il peut arriver qu'elle n'aie pas lieu certains jours ou qu'elle aie lieu avant ou après l'heure habituelle
A defaut d'abonnement, le point de vente n'a pas accès à la generation des factures, des notes de credit, des rapports périodique, des rapports journaliers et des fiches de stock
Yayo ne prend en compte que les paiements effectués à l'un de portefeuilles présents dans l'application partie Abonnement ou à un autre canal signalé par l'assistant WhatsApp dont le numéro est affiché dans l'application partie Abonnement

De l'espace publicitaire (Circulaires)

Dans l'actuel version, la liste affichée aux utilisateur est filtrée par pays
Les circulares sont accessibles et visibles pour tous les utilisateurs de Yayo dans le pays concrné, et tous peuvent créer une discussion avec le proprietaire d'un circulaire, mais seuls les proprietaires peuvent les modifier ou les supprimer
Un message envoyé dans un circualire est directement et uniquement reçu par son destinataire et créer un raccourci à la discussion sous forme de notification
Si un utilisateur est supprimé par l'administrateur de son point de vente, tous ses circulaires ne s'affichent pas aux utilisateurs de Yayo, pareil pour les messages et notifications dont il a été l'auteur

De la gestion du profile utilisateur

Il s'agit d'un espace accessible par l'utilisateur uniquement, dont il faut le mot de passe de l'utilisateur pour y acceder
Dans cette partie l'utilisateur trouve toutes les informations de son compte entre autre l'image de profile, le nom et le mot de passe et iil peut les modifier, il y trouve également le taux d'échange de son point de vente

Des annonces présentes dans Yayo

Dans le souci d'accroitre les opportunités de tous ses utilisateurs, Yayo leur affiche les annonces des entreprises et organisations dont les activités peuvent interesser les points de vente
Dans la version actuelle les annonces sont filtrées par pays
